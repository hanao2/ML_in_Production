API Documentation

Overview

This API allows you to analyze images by sending a GET request to the endpoint.

API Endpoint

The API is accessible at: http://localhost:3000/api/v1/analysis/

Request

To analyze an image, you have two options:

1. Upload it here.





2. Send a GET request with a JSON object containing a single field "uri" pointing to the image to be analyzed. 

            
{
    "uri": "path/to/your/image.jpg"
}
